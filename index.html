Percepta

Implementation of a Perceptual Computing Interface (PCI)

Jai Ratna | Manipal University Jaipur

Problem Statement

Traditional computer interaction relies on mouse & keyboard, limiting accessibility and hands-free computing.

Need: A perceptual interface that converts facial movements into digital control actions.

Proposed Solution

  • Head movement → Cursor control
  • Eyebrow raise → Click
  • Mathematical modeling → Stability
  • Tunable interaction constants

Current Features

Head-controlled cursor movement
Eyebrow raise click
Dead-zone filtering
Moving average smoothing
Exponential smoothing
Multi-constant control system
Stable landmark detection
Adjustable sensitivity

Mathematical Model

Head Movement Model:
Δx = x(t) − x(t−1)
Δy = y(t) − y(t−1)

Cursor Movement:
Cursor_X = Current_X + k_head × Δx
Cursor_Y = Current_Y + k_head × Δy

Eyebrow Click Model:
Distance = Eye_Y − Eyebrow_Y
If Distance > Threshold → Click

All interaction is controlled using tunable constants.

Important Constants

K_HEAD
EYEBROW_THRESHOLD
CLICK_COOLDOWN
DEAD_ZONE
SMOOTH_WINDOW

Future Vision

Multi-modal perceptual interface with eye gaze, voice, gestures, and facial expressions.

  • Eye gaze tracking for cursor & selection
  • Head tilt and nod for scrolling & click
  • Hand gesture navigation (swipe, zoom)
  • Mouth-based drag control
  • Facial expression-based commands
  • Voice + facial hybrid interaction
